Vision board digitální akademie

Očekávám, že akademie mě naučí digitální dovednosti a technologie, které potrebuji k tomu, abych mohla úspěšně létat v digitálním světě, orientovat se v digitálních výškách a bezpečně přistát v digitálním světě.

Technologie, se kterými se naučím pracovat:

  1. HTML
  2. CSS
  3. JavaScript
  4. React

Těším se:

  • Získat nové znalosti a praktické zkušenosti.
  • Být součástí komunity nadšených lidi.
  • Najít si nové přátele, kteří mají spolecný zájem.
  • Naprogramovat reálnou smysluplnou aplikaci.
  • Zlepšit své šance na trhu práce a najít si zajímavou a perspektivní práci v oboru.
