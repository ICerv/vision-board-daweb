Vision board digitální akademie

Ocekavam, ze akademie me nauci digitalni dovednosti a technologie, ktere potrebuji k tomu, abych mohla v digitalnim svete uspesne letat, orientovat se v digitalnich vyskach a bezpecne pristat v digitalnim svete.

Technologie, se kterymi se naucim pracovat:

  1. HTML
  2. CSS
  3. JavaScript
  4. React

Tesim se:

  • Ziskat nove znalosti a dovednosti.
  • Naprogramovat vlastni smysluplnou aplikaci.
  • Byt soucasti komunity nadsenych lidi.
  • Najit si nove pratele, kteri maji spolecny zajem.
  • Zlepsit sve sance na trhu prace a najit si zajimavou a perspektivni praci v oboru.
